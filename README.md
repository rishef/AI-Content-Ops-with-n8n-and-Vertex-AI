# AI-Content-Ops-with-n8n-and-Vertex-AI

Autonomous AI Video Generation Pipeline
n8n + Gemini + Vertex AI (VEO 3)

An end-to-end AI automation system that:

-> Generates viral short-form video ideas

-> Converts ideas into cinematic VEO prompts

-> Generates videos using Google Vertex AI (VEO 3)

-> Uploads output to Google Drive

-> Logs production lifecycle in Google Sheets

Fully automated. Zero human intervention.
